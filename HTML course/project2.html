Description

In the project the a time series test is conducted on the sample data.
The model predicts the sales made in the future using the given data. The model utilizes facebook's prophet library to conduct a time series analysis. The package is convenenient to show seasonality which is categorized daily, weekly and annualy based on data and trend.

Output

Time series prediction using the facebook's prophet package for forecasting and analysis
